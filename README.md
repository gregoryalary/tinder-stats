# TinderStats

![Tinder stats screenshot](https://user-images.githubusercontent.com/20248805/74468541-46074380-4e9b-11ea-9018-211d8a937f46.png)

## Usage

```bash
git clone "https://github.com/gregoryalary/tinder-stats"
cd tinder-stats
npm i
ng serve
```
